Autogenics Client Project

Client Project based on a Message Board for a University

Project Duration

April 19 - April 23rd, 2017 Project time-frame 5 days

Development Process

Used Agile Workflow + Trello to organize and track production

Set team norms and proper Git Workflow

Broke up features using MVC vertical slicing

Day 1

Planning

In a group setting we listed out notes that were taken at a meeting and identified which key points make up the MVP

Broke down key features into small tasks that can be done by one person

Integrated organic pair-programming for instances in which someone is struggling & seeking help

Day 2 - 3

Individually worked on vertically sliced features

Return to the kanban to identify and prioritize features + testing

Each feature included at least 1-2 written tests

Day 4 - 5

Add in extra feature upon client request

Return to the kanban to identify and prioritize features + testing

Deploy app to Heroku

Project Ownership

Produced nav bar and took over general CSS of web app

Learned how to make AJAX calls using Rails 5 and implemented across the web application.

Produced show page for individual experiements

In charge of testing, wrote all the tests to ensure the app was 91% tested

Technologies Used

Agile Workflow
